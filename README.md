# Product Management System with Content-Based Recommendations and Web Scraping

A simple yet powerful Full-Stack application for managing products, featuring an AI-driven recommendation engine and a web scraper.

---

## 🚀 Features

* **Product Management:** Add or update products in a MongoDB database.
* **Smart Recommendations:** Suggests similar products using **Cosine Similarity** (NumPy).
* **Fast Search:** Keyword-based search using MongoDB Text Index.
* **Web Scraping:** Automated data extraction from external sites using **Selenium**.

---

## 🛠️ Technologies

- **Backend:** Python (Flask)
- **Database:** MongoDB
- **Calculations:** NumPy
- **Automation:** Selenium
- **Frontend:** JavaScript, HTML, CSS
